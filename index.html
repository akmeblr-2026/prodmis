Data lives in Google Sheets as the primary database. Every save/edit/delete writes directly to your sheet. On startup the app loads from Sheets.
⚠ Common issues:
• Still shows "Not Connected" — After authorising, you must deploy a NEW VERSION (Deploy → Manage deployments → Edit → Version: New version → Deploy)
• Timeout error — Make sure "Who has access" is set to Anyone (not "Anyone with Google account")
• Wrong URL — URL must end in /exec, not /dev
